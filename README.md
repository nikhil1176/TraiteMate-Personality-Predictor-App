# TraiteMate 🧠
**Discover Yourself with Your Personality Insights.**

TraiteMate is a full-stack MERN application designed to provide users with deep personality analysis through interactive quizzes and data-driven results.

## 🚀 Features
- **Interactive Quiz:** Engaging user interface for personality assessment.
- **Real-time Results:** Instant feedback and trait analysis.
- **Secure Auth:** JWT-based authentication for user profiles.
- **Cloud Database:** Powered by MongoDB Atlas.

## 🛠️ Tech Stack
- **Frontend:** React.js, CSS3, FontAwesome
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)

## ⚙️ Installation & Setup
1. Clone the repo: `git clone https://github.com/nikhil1176/TraiteMate.git`
2. Install Backend dependencies: `cd backend && npm install`
3. Install Frontend dependencies: `cd frontend && npm install`
4. Create a `.env` file in the `/backend` folder with your `MONGO_URI`.
5. Run the app: 
   - Backend: `npm start` (in /backend)
   - Frontend: `npm run dev` (in /frontend)
